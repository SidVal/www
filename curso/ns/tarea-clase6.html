Manipulación del DOM

Paso 1: Obtener un elemento

  • document.querySelector('selector'): Es uno de los métodos para obtener un elemento. Devuelve sólo el primer elemento que coincida con el selector CSS dado
  • document.querySelector('selector'): Obtiene un grupo de elementos. Devuelve un array con todos los elementos que coincidan con el selector CSS dado
  • document.getElementById(''): obtiene un solo elemento unicamente a través de su ID

Paso 2: Manipular el elemento una vez obtenido:

  • CSS: elemento.style.propiedadCss
  • HTML: elemento.textContent
  • HTML: elemento.innerHTML

Manipulación de DOM a través de un evento - Intro:

Tambien se puede manipular un elemento a través de un evento. Repasemos antes... Un evento es la interacción del usuario con nuestra página Web. Esta interacción activa una función.

Uno de los eventos más utilizados es el evento click. La estrucutra sintaxica para un evento es el siguiente

elemento.addEventListener("evento",funcion a ejecutar)
